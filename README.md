# Prototype-5-unity
Dans ce protoype, vous programmerez un jeu pour tester les réflexes du joueur, où le but est de cliquer et de détruire des objets lancés au hasard en l'air avant qu'ils ne tombent de l'écran. En créant ce prototype, vous apprendrez à implémenter une interface utilisateur - ou UI - dans vos projets. Vous ajouterez un écran titre avec un menu de sélection de difficulté qui contrôlera la difficulté du gameplay, vous ajouterez un affichage de score qui suivra le nombre de points que le joueur a gagnés, et vous ajouterez un écran Game Over, qui permettra au lecteur pour redémarrer et réessayer. En apprenant ces compétences, vous serez en mesure de créer une expérience entièrement "jouable" dont l'utilisateur pourra profiter du début à la fin sans avoir à redémarrer l'application pour l'essayer à nouveau.
